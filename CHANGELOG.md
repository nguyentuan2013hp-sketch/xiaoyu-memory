### 📖 核心规则

#### 四大操作
| 操作 | 触发条件 | 操作内容 |
|------|---------|---------|
| ADD | 发现新模式 | 创建记录，标记 pending |
| MODIFY | 效果不佳 | 更新策略，重置验证 |
| DELETE | 验证失败 ≥3次 | 标记 deprecated |
| KEEP | 稳定有效 | 提升置信度，标记 verified |

#### 验证状态
| 状态 | 含义 | 标准 |
|------|------|------|
| untested | 未验证 | 首次创建 |
| pending | 验证中 | validations < 5 |
| verified | 已验证 | success_rate ≥ 80% |
| deprecated | 已废弃 | 删除或大幅修改 |

#### 版本号规则
```
主版本.次版本.修订号
- 修订号 (x.x.1): 小修改、错误修复
- 次版本 (x.1.x): 新增功能、框架结构变更
- 主版本 (1.x.x): 重大变更、不兼容改动
```

#### 迭代输出规则
- 每次迭代更新 CHANGELOG.md
- 记录：迭代内容 + 规则变更
- 输出：版本号 + 日期 + 内容 + 规则
