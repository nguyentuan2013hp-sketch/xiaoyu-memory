## 📋 阶段1: 策划准备 (准备期: 1-2小时)

### 步骤1.1: 主题确定
```
输入: 创意灵感/观众需求/趋势分析
工具: P-005 内容创意提示词
输出: 3-5个候选主题
```

**检查点**:
- [ ] 主题是否有明确观众价值？
- [ ] 是否解决具体痛点或需求？
- [ ] 是否有差异化创新点？
- [ ] 是否适合目标平台？

### 步骤1.2: 目标设定
```
输入: 选定主题
工具: SMART目标框架
输出: 明确的内容目标
```

**目标维度**:
1. **教育目标**: 观众学到什么？
2. **情感目标**: 观众感受什么？
3. **行动目标**: 观众做什么？
4. **业务目标**: 实现什么效果？

### 步骤1.3: 观众分析
```
输入: 主题 + 目标
工具: 观众画像模板
输出: 详细观众画像
```

**分析要素**:
- 人口统计学特征
- 心理特征和价值观
- 内容消费习惯
- 痛点和未满足需求
- 语言风格偏好

### 步骤1.4: 竞品研究
```
输入: 主题 + 观众分析
工具: P-004 爆款分析提示词
输出: 竞品分析报告
```

**分析内容**:
- 成功内容模式提取
- 失败内容教训总结
- 市场机会识别
- 差异化策略制定

---

## 🎨 阶段2: 内容设计 (设计期: 2-3小时)

### 步骤2.1: 标题创作
```
输入: 主题 + 观众分析 + 竞品研究
工具: TP模式库 + P-001提示词
输出: 3-5个标题选项
```

**创作流程**:
```
1. 确定标题类型 (问题解决/情感共鸣/清单实用)
2. 选择TP模式 (TP-001/TP-002/TP-003等)
3. 使用P-001生成多个选项
4. A/B测试选择最佳标题
```

**质量检查**:
- [ ] 是否包含明确价值主张？
- [ ] 是否引发好奇心或情感？
- [ ] 是否适合目标观众？
- [ ] 是否优化SEO关键词？

### 步骤2.2: 大纲设计
```
输入: 选定标题 + 内容目标
工具: F格式库 + P-002提示词
输出: 完整内容大纲
```

**设计流程**:
```
1. 选择F格式 (基于内容类型)
2. 确定时间分配比例
3. 设计情感发展曲线
4. 规划关键转折点
```

**结构要素**:
- 开场钩子 (0:00-0:30)
- 主要内容 (3-4个逻辑部分)
- 转折深化 (中点认知升级)
- 结尾升华 (价值总结+行动号召)

### 步骤2.3: 元素填充
```
输入: 大纲框架
工具: E元素库 + 内容素材
输出: 丰富的内容脚本
```

**元素应用**:
```
每个大纲部分:
1. 选择2-3个E元素组合
2. 准备具体案例和数据
3. 设计视觉和音频配合
4. 规划互动时刻
```

**元素组合策略**:
- 开场: E-001 + E-004 (故事+情感)
- 发展: E-002 + E-003 (数据+案例)
- 深化: E-005 + E-007 (对比+问答)
- 结尾: E-006 + E-008 (清单+隐喻)

### 步骤2.4: 脚本撰写
```
输入: 大纲 + 元素填充
工具: P-003脚本优化提示词
输出: 完整拍摄脚本
```

**脚本要素**:
- 逐分钟内容要点
- 视觉元素描述
- 音频效果提示
- 主持人动作指导
- 屏幕文字设计

**优化重点**:
1. 语言口语化自然化
2. 节奏变化和停顿
3. 情感表达强化
4. 关键信息突出

---

## 🎬 阶段3: 制作发布 (执行期: 3-8小时)

### 步骤3.1: 视觉设计
```
输入: 完整脚本
工具: 视觉规划模板
输出: 视觉制作清单
```

**视觉元素**:
1. **拍摄场景**: 背景、灯光、道具
2. **镜头设计**: 景别、角度、运动
3. **画面元素**: 文字、图形、动画
4. **色彩风格**: 调色、滤镜、氛围

### 步骤3.2: 音频设计
```
输入: 脚本 + 视觉设计
工具: 音频规划模板
输出: 音频制作清单
```

**音频元素**:
1. **人声录制**: 语调、节奏、情感
2. **背景音乐**: 情绪匹配、音量控制
3. **音效设计**: 强调、过渡、氛围
4. **音频处理**: 降噪、均衡、压缩

### 步骤3.3: 拍摄制作
```
输入: 所有设计文档
工具: 拍摄清单 + 制作软件
输出: 原始素材文件
```

**制作流程**:
```
准备 → 拍摄 → 整理 → 备份
检查点:
- 设备检查
- 环境准备
- 表演排练
- 素材管理
```

### 步骤3.4: 后期编辑
```
输入: 原始素材
工具: 编辑软件 + 编辑模板
输出: 成品视频文件
```

**编辑流程**:
```
1. 粗剪: 结构搭建
2. 精剪: 节奏优化
3. 包装: 视觉音频完善
4. 审核: 质量检查
```

**质量检查清单**:
- [ ] 前30秒是否足够吸引？
- [ ] 节奏是否变化丰富？
- [ ] 关键信息是否突出？
- [ ] 情感表达是否到位？
- [ ] 技术质量是否达标？

### 步骤3.5: 发布优化
```
输入: 成品视频
工具: 发布清单模板
输出: 多平台发布内容
```

**发布要素**:
1. **标题优化**: A/B测试最终标题
2. **描述撰写**: SEO优化 + 价值说明
3. **标签设置**: 关键词策略
4. **缩略图设计**: 点击率优化
5. **发布时间**: 观众活跃时段

### 步骤3.6: 互动管理
```
输入: 发布后数据
工具: 互动管理模板
输出: 观众关系维护
```

**互动策略**:
1. **评论回复**: 及时、真诚、有价值
2. **问题收集**: 观众反馈和疑问
3. **社区建设**: 观众连接和归属感
4. **内容迭代**: 基于反馈优化

---

## 📊 阶段4: 数据分析 (持续优化)

### 步骤4.1: 数据收集
```
输入: 发布内容
工具: 数据分析平台
输出: 完整表现数据
```

**关键指标**:
1. **观看数据**: 播放量、完播率、留存曲线
2. **互动数据**: 点赞、评论、分享、收藏
3. **转化数据**: 点击率、行动采取、转化率
4. **观众数据**:  demographics、观看设备、来源

### 步骤4.2: 效果分析
```
输入: 原始数据
工具: P-004分析提示词 + 数据分析
输出: 效果分析报告
```

**分析维度**:
1. **内容效果**: 哪些部分最有效？
2. **观众反应**: 情感共鸣点在哪里？
3. **传播机制**: 为什么被分享？
4. **改进机会**: 哪些可以做得更好？

### 步骤4.3: 模式提取
```
输入: 效果分析
工具: 模式识别框架
输出: 可复制成功模式
```

**提取内容**:
- 成功的TP/E/F/P组合
- 有效的情感策略
- 优化的节奏设计
- 观众偏好洞察

### 步骤4.4: 系统迭代
```
输入: 新模式和经验
工具: 模式库更新流程
输出: 优化的模式库
```

**迭代流程**:
```
数据分析 → 模式提取 → 验证测试 → 
模式库更新 → 创作应用 → 再次分析
```

## 🔄 快速创作流程 (简化版)

### 30分钟快速创作
```
1. 主题确定 (5分钟)
   - 使用P-005生成创意
   - 选择最有潜力的主题

2. 标题创作 (5分钟)
   - 使用TP模式库参考
   - 使用P-001生成标题

3. 大纲设计 (10分钟)
   - 选择F-001或F-004格式
   - 使用P-002生成大纲

4. 脚本要点 (10分钟)
   - 使用E-001/E-004/E-006组合
   - 撰写关键内容要点
```

### 2小时标准创作
```
包含完整阶段2的所有步骤
适合重要内容或系列内容
```

### 1天深度创作
```
包含所有阶段1-3的完整流程
适合旗舰内容或品牌内容
```
