## 🎯 P 提示词分类

### P-001: 标题生成提示词
**用途**: 基于主题生成爆款标题

**基础模板**:
```
你是一个专业的YouTube内容创作者，擅长创作吸引人的标题。

主题: [输入主题]
目标观众: [输入观众特征]
情感基调: [输入情感基调]

请基于以下要求生成5个标题选项:
1. 使用TP模式库中的有效模式
2. 包含数字或具体承诺
3. 引发好奇心或情感共鸣
4. 适合目标观众偏好
5. 优化SEO关键词

请为每个标题提供:
- 使用的TP模式
- 预期效果分析
- 适合的内容类型
```

**高级变体**:
```
基于[主题]和以下数据生成标题:
- 热门关键词: [关键词列表]
- 竞争对手标题: [参考标题]
- 观众痛点: [痛点描述]
- 独特卖点: [USP]

生成3个不同风格的标题:
1. 问题解决型 (使用How to模式)
2. 情感共鸣型 (使用故事模式)
3. 清单实用型 (使用数字清单模式)

每个标题附带:
- 情感触发点
- 好奇心钩子
- 价值承诺
- SEO优化建议
```

**示例应用**:
```
主题: 提高工作效率
目标观众: 办公室白领，25-35岁
情感基调: 实用、激励、解决痛点

生成结果:
1. "如何每天多出2小时高效时间" (TP-002)
2. "拖延症正在偷走你的未来" (警示型)
3. "5个科学方法让你的效率提升300%" (清单型)
```

### P-002: 大纲生成提示词
**用途**: 基于标题生成完整内容大纲

**基础模板**:
```
基于标题"[标题]"创作一个YouTube视频大纲。

使用F格式库中的[格式类型]结构，包含以下部分:

1. 开场钩子 (0:00-0:30)
   - 使用E-001或E-004元素
   - 明确价值主张
   - 引发继续观看

2. 主要内容 (分3-4个部分)
   - 每个部分使用不同的E元素
   - 逻辑递进关系
   - 包含具体案例或数据

3. 结尾升华
   - 总结核心价值
   - 行动号召
   - 互动引导

请提供:
- 时间分配建议
- 视觉元素建议
- 转折点设计
- 观众留存策略
```

**高级变体**:
```
为标题"[标题]"创建详细大纲，基于以下分析:

内容深度分析:
- 核心信息: [主要观点]
- 支持论据: [数据/案例]
- 情感主线: [情感发展]
- 认知转折: [观念改变]

结构优化要求:
- 前30秒留存率>70%
- 中点转折强化认知
- 结尾提供明确价值
- 适合[平台]传播

生成包含:
1. 逐分钟脚本要点
2. 视觉和音频建议
3. 互动时刻设计
4. 分享触发点
```

**示例应用**:
```
标题: "如何建立早晨习惯系统"

生成大纲:
1. 开场 (0:00-0:45)
   - 故事: 我曾经早晨混乱的经历
   - 痛点: 早晨决定全天效率
   - 承诺: 5步系统解决方案

2. 主体 (0:45-7:30)
   - 部分1: 环境设计 (E-002数据支撑)
   - 部分2: 仪式建立 (E-003案例引用)
   - 部分3: 习惯叠加 (E-006清单结构)
   - 部分4: 调整优化 (E-005对比反差)

3. 结尾 (7:30-8:30)
   - 结果展示: 我的转变
   - 行动号召: 30天挑战
   - 互动: 分享你的早晨习惯
```

### P-003: 脚本优化提示词
**用途**: 优化现有脚本，提高内容质量

**基础模板**:
```
优化以下YouTube脚本，提高观众留存和互动:

原始脚本:
[粘贴脚本]

优化要求:
1. 强化开场钩子 (使用E-004情感共鸣)
2. 增加数据支撑 (添加E-002元素)
3. 改善节奏变化 (每2-3分钟有高潮)
4. 增强结尾号召 (明确行动步骤)
5. 优化语言表达 (更生动、具体)

请提供:
- 优化版本脚本
- 修改说明和理由
- 预期效果提升
- 进一步优化建议
```

**高级变体**:
```
深度优化脚本，基于观众心理学:

脚本分析:
- 情感曲线: [当前情感发展]
- 认知负荷: [信息密度分析]
- 记忆点: [关键信息突出度]
- 行动障碍: [转化阻力点]

优化策略:
1. 前30秒: 建立强烈情感连接
2. 2分钟点: 提供第一个"aha时刻"
3. 中点: 认知转折和深化
4. 结尾: 情感升华和行动催化

生成:
- 情感优化版本
- 节奏调整建议
- 视觉配合方案
- A/B测试建议
```

**示例应用**:
```
优化前: "今天讲提高效率的方法..."
优化后: "你是否感觉每天忙忙碌碌却一事无成？我曾经也是这样，直到发现了这个系统..."

优化点:
1. 开场从陈述改为问题+个人故事
2. 增加具体数据: "效率提升3倍"
3. 添加视觉隐喻: "注意力像肌肉"
4. 强化结尾号召: "从明天早晨开始"
```

### P-004: 爆款分析提示词
**用途**: 分析爆款内容，提取可复制模式

**基础模板**:
```
分析以下爆款内容，提取可复制的创作模式:

内容链接: [URL或描述]
播放量: [数据]
互动数据: [点赞、评论、分享]

请分析:
1. TP模式分析 (标题使用了什么模式)
2. E元素分析 (内容包含哪些元素)
3. F格式分析 (整体结构是什么格式)
4. 情感策略 (如何建立情感连接)
5. 价值传递 (如何提供明确价值)

提取:
- 3个最有效的模式
- 2个创新点
- 1个可立即应用的技巧
- 改进建议 (如果是你创作会如何优化)
```

**高级变体**:
```
进行深度爆款内容逆向工程:

内容数据:
- 留存曲线分析
- 互动时间点分布
- 评论情感分析
- 分享动机研究

分析维度:
1. 注意力经济学: 如何分配观众注意力
2. 情感工程学: 情感发展和转折设计
3. 认知心理学: 信息处理和记忆强化
4. 行为设计学: 行动触发和转化优化

生成:
- 完整模式提取报告
- 可复制模板
- 平台适配建议
- 风险预警和规避
```

**示例应用**:
```
分析: Matt D'Avella "数字极简主义"视频

提取模式:
1. TP模式: 对比反差型标题
2. E元素: 故事开场+数据支撑+视觉隐喻
3. F格式: 旅程式结构
4. 情感策略: 从焦虑到解放的情感弧线
5. 价值: 具体可操作的清理步骤

可复制技巧:
- 使用"前后对比"展示价值
- "数字垃圾"视觉隐喻
- 30天挑战框架
```

### P-005: 内容创意提示词
**用途**: 基于趋势和观众需求生成内容创意

**基础模板**:
```
基于以下输入生成10个内容创意:

主题领域: [领域]
目标观众: [观众描述]
当前趋势: [趋势关键词]
竞争对手分析: [竞品内容]

创意要求:
1. 结合TP/E/F模式库
2. 解决具体观众痛点
3. 有差异化创新点
4. 适合[平台]传播
5. 有系列化潜力

为每个创意提供:
- 核心价值主张
- 目标情感反应
- 预期播放量范围
- 制作难度评估
- 系列化扩展思路
```

**高级变体**:
```
进行系统性内容机会挖掘:

市场分析维度:
- 搜索趋势和关键词缺口
- 观众未满足需求
- 竞品内容弱点
- 平台算法偏好
- 文化和社会趋势

创意生成策略:
1. 问题解决型: 直接解决观众痛点
2. 情感共鸣型: 建立深度情感连接
3. 思想领导型: 提供新视角和洞察
4. 娱乐价值型: 提供轻松有趣内容
5. 实用工具型: 提供具体方法和工具

输出:
- 机会矩阵分析
- 优先级排序
- 资源需求评估
- 风险收益分析
```

### P-007: A/B测试设计提示词
**用途**: 设计内容A/B测试方案

**基础模板**:
```
为以下内容设计A/B测试方案:

测试内容: [内容描述]
测试目标: [提高完播率/互动率/分享率]

设计3个测试变量:
1. 标题变体 (测试不同TP模式)
2. 开场变体 (测试不同E元素)
3. 结构变体 (测试不同F格式)

为每个变体提供:
- 具体实施方案
- 预期效果假设
- 数据收集方法
- 成功标准定义
- 决策逻辑框架
```

### 最佳实践总结

#### 标题生成 (P-001):
- **关键**: 明确价值主张 + 情感钩子
- **技巧**: 结合数字 + 具体承诺
- **避免**: 模糊语言，缺乏针对性

#### 大纲生成 (P-002):
- **关键**: 逻辑结构 + 节奏控制
- **技巧**: 每部分明确价值点
- **避免**: 信息过载，结构混乱

#### 脚本优化 (P-003):
- **关键**: 情感强化 + 具体化
- **技巧**: 添加故事和数据
- **避免**: 抽象论述，缺乏例证

## 🎨 提示词使用指南

### 基础使用流程

#### 步骤1: 明确需求
```
1. 确定内容类型 (教程/故事/观点)
2. 分析目标观众 (需求/痛点/偏好)
3. 设定内容目标 (教育/娱乐/启发)
4. 选择合适提示词类型
```

#### 步骤2: 定制输入
```
关键输入要素:
- 主题和核心信息
- 观众特征和痛点
- 情感基调和风格
- 平台特性和限制
- 竞争对手参考
```

#### 步骤3: 迭代优化
```
优化循环:
生成 → 评估 → 调整 → 再生成
评估标准:
- 是否符合TP/E/F模式
- 是否有情感共鸣
- 是否提供明确价值
- 是否适合目标平台
```
