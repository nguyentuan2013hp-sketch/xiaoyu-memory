# 视频分析工作流 v1.0

## 阶段一：准备阶段 (5分钟)
1. **视频选择**：选择高价值、可学习的视频
2. **工具准备**：浏览器、笔记工具、分析模板
3. **目标设定**：明确本次分析的学习重点

## 阶段二：观看与分析 (15-20分钟)
### 第一遍观看：整体感受
- 记录第一印象
- 识别核心价值主张
- 标记关键时间点

### 第二遍观看：5维分析
1. **标题设计**：吸引力、关键词、情感价值
2. **开头钩子**：前30秒的吸引力设计
3. **内容结构**：逻辑流程、章节划分、节奏控制
4. **制作技巧**：视觉、音频、编辑、转场
5. **行动号召**：结尾转化设计、社区互动

## 阶段三：结构化输出 (10分钟)
1. **填写分析模板**：5维评分+详细分析
2. **提取可学习模式**：3-5个核心技巧
3. **创新应用建议**：如何应用到自己的创作
4. **保存到记忆系统**：建立可搜索的知识库

## 阶段四：回顾与进化 (5分钟)
1. **分析质量检查**：本次分析的优缺点
2. **框架改进**：优化分析模板或工作流
3. **下次优化点**：记录需要改进的地方

## 量化指标
- **分析用时**：目标35-40分钟/视频
- **分析深度**：5个维度全覆盖
- **输出质量**：可立即应用的学习点
- **进化速度**：每次分析都比上次更好

## 工具清单
1. 浏览器（视频播放）
2. 飞书文档（记录分析）
3. 分析模板（标准化输出）
4. 记忆系统（知识积累）
