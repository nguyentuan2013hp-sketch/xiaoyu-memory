## 🔧 技术配置

### 唤醒词设置
```yaml
wake_words:
  primary: "龙虾"      # 主要唤醒词
  secondary: "OpenClaw" # 备用唤醒词  
  chinese: "小玉"      # 中文唤醒词
  
detection:
  sensitivity: "high"   # 高灵敏度
  language: "zh-CN"     # 中文检测
  continuous: true      # 持续监听
```

### TTS语音配置
```yaml
tts:
  engine: "sag"                    # ElevenLabs TTS
  voice: "周先森克隆声音"           # 克隆声音
  speed: 1.0                      # 正常语速
  emotion: "professional"         # 专业语气
  
output:
  format: "mp3"                   # 输出格式
  quality: "high"                 # 高质量
  auto_play: true                 # 自动播放
```

### 语音识别配置
```yaml
speech_recognition:
  engine: "whisper"               # OpenAI Whisper
  language: "zh"                  # 中文识别
  realtime: true                  # 实时识别
  
processing:
  auto_transcribe: true           # 自动转录语音消息
  save_transcripts: true          # 保存转录文本
  context_aware: true             # 上下文感知
```

## 🚨 故障处理

### 常见问题
1. **TTS失败**: 回退到文字回复
2. **唤醒检测失败**: 手动触发模式
3. **语音识别失败**: 请求重发或文字输入
4. **网络问题**: 本地缓存，稍后同步

### 恢复机制
1. 自动重试 (最多3次)
2. 降级到文字模式
3. 记录错误日志
4. 通知用户状态
